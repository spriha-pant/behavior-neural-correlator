# Phase 1 — Stage 1: Linear Regression | Binary Lick Detection
# =============================================================

## Project Structure

```
project/
├── config.yaml               ← All settings live here
├── requirements.txt
├── data/
│   ├── raw/                  ← PUT YOUR CSV FILE HERE
│   └── processed/            ← Auto-generated by 2_preprocess.py
├── src/
│   ├── 1_load_data.py        ← Sanity check your data
│   ├── 2_preprocess.py       ← Feature engineering + train/test split
│   ├── 3_train_model.py      ← Train Linear + Logistic Regression
│   ├── 4_evaluate.py         ← Metrics, confusion matrix, plots
│   └── 5_predict.py          ← Run model on new unseen data
├── models/                   ← Saved model files (auto-generated)
└── results/                  ← Plots and CSVs (auto-generated)
```

## Setup (run once)

```bash
pip install -r requirements.txt
mkdir -p data/raw data/processed models results
```

## How to Run (in order, from the project/ root folder)

```bash
# Step 1: Check your data loads correctly
python src/1_load_data.py

# Step 2: Clean + engineer features + split
python src/2_preprocess.py

# Step 3: Train the models
python src/3_train_model.py

# Step 4: Evaluate and generate plots
python src/4_evaluate.py

# Step 5 (optional): Predict on a new CSV file
python src/5_predict.py data/raw/your_new_file.csv logistic
```

## Output files (in results/)
- `stage1_metrics.csv`               — Accuracy, Precision, Recall, F1 table
- `stage1_confusion_matrices.png`    — Confusion matrix for both models
- `stage1_metric_comparison.png`     — Side-by-side bar chart
- `stage1_prediction_timeline.png`   — Predicted vs actual over time
- `predictions_<filename>_<model>.csv` — From 5_predict.py

## Important notes
- Always run scripts from the `project/` root directory, not from inside `src/`
- Never edit files in `data/raw/` — that's your ground truth
- If you change feature engineering in `2_preprocess.py`, re-run steps 2→3→4
